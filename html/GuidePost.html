Wochenaufgabe Wanderwegweiser

Ziel

Die Ziele der Wochenaufgabe findet Ihr im blog

OSMCount zählt für die Wanderwegweiser die unterschiedlichen Taggings mit ihren Subtags. Das sind:

  • Nodes mit information=guidepost
  • Relationen vom type=destination_sign
  • Ways, die mit destination Tags versehen sind

    • Vorgaben

    Vorgaben für diese Aufgabe gibt es natürlich nicht.

    Istwerte

    Es werden Queries vom folgenden Typ benutzt: 
    tbd.

    SubTags

    Für die Nodes werden die folgenden Subtags gezählt:

    • hiking
    • bicycle
    • name
    • ref
    • operator
    • material
    • colour
    • "note:destination"
    • inscription
    • description
    Für die Relation destination_sign werden die folgenden Subtags genommen:
    • destination
    • time
    • distance

    Die Zahlen können absolut, oder als Prozentangabe zu den Nodes angezeigt werden.
