Wochenaufgabe Wanderwegweiser

Ziel

Die Ziele der Wochenaufgabe findet Ihr im blog

OSMCount zählt für die Wanderwegweiser die unterschiedlichen Taggings mit ihren Subtags. Das sind:

  • Nodes mit information=guidepost
  • Relationen vom type=destination_sign
  • Ways, die mit destination Tags versehen sind
    Diese Abfrage ist noch "experimentell", sie schluckt so viel Overpass time, das die Auswertung instabil ist.

    • Vorgaben

    Vorgaben für diese Aufgabe gibt es natürlich nicht.

    Istwerte

    Es werden Queries vom folgenden Typ benutzt: 
    tbd.

    SubTags

    Für die Nodes werden die folgenden Subtags gezählt:

    • hiking
    • bicycle
    • name
    • ref
    • operator
    • material
    • colour
    • "note:destination"
    • inscription
    • description
    Für die Relation destination_sign werden die folgenden Subtags genommen:
    • destination
    • time
    • distance
    Für die highway destination:* Tags werden die folgenden Subtags genommen:
    • destination:f/b -> destination:forward,destination:backward
    • destination:symbol:f/b -> destination:symbol:forward,destination:symbol:backward
    • destination:ref:f/b -> destination:ref:forward,destination:ref:backward
    • destination:lang:f/b -> destination:lang:forward,destination:lang:backward
    • destination:NESW -> destination:N,destination:NE...

    Die Zahlen können absolut, oder als Prozentangabe zu den Nodes angezeigt werden.
