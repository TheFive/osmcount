OSMCount Handbuch

Die Menüführung

Die Menüleiste wird immer angezeigt. Ein Klick auf das OSMCount Logo führt auf die Startseite.

Neben dem Logo befinden sich die folgenden Menüs.

  • Aufgaben
    Liste der Aufgaben mit kurzen Beschreibungen
  • Tabelle
    Dreh- und Angelpunkt von OSM Count, hier gibt es die Auswertungen
  • POI
    Experimentell, hier sollen in Zukunft aktuelle POI Daten von den unterschiedlichen Aufgaben gezeigt werden, wir zur Zeit eher sporadisch aktualisiert.

Im rechten Bereich der Menüleiste gibt es die kontextabhängigen Menüs. Dies sind:


  • Mit dem Grafiksymbol kann die Plotly Komponente aufgerufen werden

  • Mit dem Exportsymbol wird der CSV Export aufgerufen

  • Das Informationszeichen springt zur Kurzbeschreibung der Aufgabe

  • Das Zahnrad ruft unterschiedliche Funktionen auf. Dies die Konfiguration der Tabelle, oder aber auch die Aufhebung eines Fehlers

Einstellung der Tabellensicht

Die Tabellensicht kann über das Icon eingestellt werden. Probiert bitte die unterschiedlichen Einstellungen aus. Eingestellt werden kann:

  • Der Ort der Tabelle
    Wurde die Tabelle durch Klick auf einen Schlüssel auf ein bestimmtes Gebiet eingeschränkt, so kann hier die Einschränkung stufenweise aufgehoben werden.
  • Anzeige des Feldes
    Sollen nicht alle Objekte, sondern nur die mit einem bestimmten Tag (Subtag) dargestellt werden, so kann das hier ausgewählt werden.
  • Prozentanzeige
    Falls die Anzeige in Prozent (von der Gesamtsumme oder Vorgabe) erfolgen soll
  • Zeitperiode
    Ist die Zeitachse in Tag, Monat oder Jahr aufzuteilen
  • Anzeige Seit
    Bei Bedarf kann hier eine zu lange Historie ausgeblendet werden
  • Anzahl der Stellen für den Schlüssel
    OSM Count basiert auf dem "Schlüssel", über dessen Länge, die Informationen (Land, Bundesland, Gemeinde...) aufgeklappt werden können. Die Schlüssellänge korrespondiert aber nicht immer konkret mit einer Verwaltungsstufe

Die Tabelle

Die Tabelle stellt in der ersten Spalte den Schlüssel dar. Dies ist z.B. der offizielle Gemeindenummernschlüssel, oder aber eine Kombination aus verschiedenen Schlüsseln. Klickt man auf einen solchen so werden nur noch Daten angezeigt, die mit diesem Schlüssel beginnen.
In der zweiten Spalte wird der Name (entnommen aus dem OSM Name Tag) des Gebietes dargestellt. Die weiteren Spalten sind dann zeitbezogen. Dabei wird aus der angegebenen Zeit immer der aktuellste Wert genommen. Läuft eine Abfrage noch, so kann so die aktuellste Spalte unvollständig sein.
Die Spalte Service hat die folgenden kurzen Links:

  • O
    Overpass Abfrage, wird nur angezeigt, wenn für die Zeile eine Overpass Abfrage ausgeführt wurde.
  • R
    Run Overpass Abfrage, führt die Overpass Abfrage direkt aus

Im unteren Bereich der Tabelle werden die Informationen gibt es die folgenden Informationen.
  • Offene Queries
    Gibt an, wie viele Queries für diese Tabelle noch an Overpass gesendet werden müssen. Solange dies nicht 0 ist, fehlen in der aktuellen Spalte Daten.
  • Fehler
    Gibt die Fehler für die Query an. Bitte anklicken und über das Zahnrad Symbol die Fehler resetten, damit OSMCount diese erneut an die Overpass sendet.
